# Thermal Process Analyzer

Aplikasi Python untuk analisis data retort dan heat penetration study.

## Fitur

* Hitung F0 otomatis
* Deteksi Cold Spot
* Deteksi Hot Spot
* Perhitungan Lethal Rate (LR)
* F0 Heating, Holding, Cooling
* Perhitungan fh dan jh
* Export laporan Excel
* Grafik suhu dan F0 kumulatif

## Input

File CSV dengan format:

waktu,DL1,DL2,DL3
0,75,76,74
1,80,81,79
2,90,91,89

## Output

* Summary
* Data Retort
* F0 Logger
* Kurva Suhu
* Kurva F0
* File Excel Laporan
